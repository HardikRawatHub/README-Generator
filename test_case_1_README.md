## 📖 Description

An automated trading bot for cryptocurrency markets using sentiment analysis and ML models.

## ✨ Features

- ✅ Real-time sentiment analysis
- ✅ Multiple exchange support
- ✅ Backtesting capabilities
- ✅ Risk management system

## 🚀 Installation

1. Clone the repo: git clone https://github.com/hardikrawat/crypto-trading-bot
2. Install dependencies: pip install -r requirements.txt
3. Set up API keys in .env file
4. Run: python bot.py

## 💡 Usage

```bash
python bot.py --exchange binance --pair BTC/USDT
```

```bash
python bot.py --strategy sentiment --interval 1h
```
