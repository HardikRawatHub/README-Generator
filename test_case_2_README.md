## 📖 Description

Analysis of trader behavior on Hyperliquid DEX using Fear & Greed Index and ML clustering.

## ✨ Features

- ✅ Fear & Greed Index integration
- ✅ KMeans trader clustering
- ✅ Gradient Boosting predictions
- ✅ Interactive Streamlit dashboard

## 🚀 Installation

1. git clone https://github.com/hardikrawat/hyperliquid-sentiment
2. pip install -r requirements.txt
3. jupyter notebook analysis.ipynb

## 💡 Usage

```bash
streamlit run dashboard.py
```

```bash
python analysis.py --date 2024-01-01
```
