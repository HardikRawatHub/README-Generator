# README Generator

A simple Python CLI tool that collects project details interactively and generates a clean `README.md` file.

## Features

- Interactive prompts for project metadata
- Auto-generates badges (Python, License, GitHub stars/forks)
- Supports custom:
  - Installation steps
  - Usage examples
  - Feature list
- Writes output directly to `README.md`
