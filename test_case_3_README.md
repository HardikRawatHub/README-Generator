## 📖 Description

A CLI tool that automatically generates professional README.md files based on user input.

## ✨ Features

- ✅ Interactive CLI interface
- ✅ Auto badge generation
- ✅ Custom sections support
- ✅ 3 built-in test cases

## 🚀 Installation

1. git clone https://github.com/hardikrawat/readme-generator
2. cd readme-generator
3. pip install -r requirements.txt
4. python readme_generator.py

## 💡 Usage

```bash
python readme_generator.py
```

```bash
python readme_generator.py --test
```
